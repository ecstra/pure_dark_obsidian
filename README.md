# Pure Dark for Obsidian

A clean, highly customized Obsidian theme designed to achieve perfect 1-to-1 visual parity with exported PDFs. Built with a focus on beautiful typography, structured headings, and seamless readability in both Light and Dark modes.

## Key Features

This theme overhauls the default Obsidian UI with careful attention to document structure and coding aesthetics:

* **True Light & Dark Modes:** Carefully selected color palettes for both modes, ensuring contrast and readability without eye strain.
* **Custom Typography:** Integrates *Montserrat*, *Open Sans*, and *JetBrains Mono* globally across the app to replace the default fonts.
* **Advanced Code & Syntax Highlighting:** Tailored syntax token colors for code blocks that adapt perfectly to both light and dark backgrounds.
* **Enhanced Headings:** * Centered `H1` titles with a decorative ✦ accent.
  * `H3` through `H6` feature custom left-border accents and subtle background gradients to create a clear visual hierarchy.
* **Stylized Elements:** Custom checkbox styling, pill-shaped tags, zebra-striped tables, and redesigned footnote sections.
* **Custom Callouts:** Full support for Obsidian's native callouts (`[!info]`, `[!danger]`, `[!example]`, etc.) with bespoke background opacities and border colors.
